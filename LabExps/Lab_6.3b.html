Laboratory Exercise 6.3b (continued)
We continue with a 1.5 second sample of a Poisson-distributed random process whose probability mass function p(n|λ) = p(x) = λne–λ/n! where λ > 0 and n ≥ 0. The process has a single parameter λ randomly chosen from the set {0.1, 0.2, 0.4, 0.8, 1.6, 3.2, 6.4, 12.8, 25.6}. Each time you perform this exercise by pressing the “retry” button below () another sample of the process will be generated but with a new, probably different value for the parameter λ.
  1. Using the expression f, N, and the measured value φ0 estimate the value of λ that was used to generate this sample. Call this estimate λf.

  2. Using the expression g, N, and the measured value S0 as well as the measured φ0 estimate the value of λ that was used to generate this sample. Call this estimate λg.

  3. By what percentage do λf and λg differ from each other? By what percentage do they differ from the true value λ?

  4. Can you hear any difference between the exponential and Poisson noise signals?

  5. Do these two random signals—e[n] and/or P[n]—meet the definition of white noise as presented in Example: Is that coin fair?.

  6. Could you justify processing any of these noise signals (from this or the previous exercise)—b(t), e(t), or P(t)—with algorithms designed for white noise? And, if so, under what conditions?
