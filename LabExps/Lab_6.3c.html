Laboratory Exercise 6.3c (conclusion)
In the previous sections you have shown that with N, φ0, and S0, there are two different ways to estimate the single parameter associated with an exponential distribution, α or the Poisson distribution, λ. In this problem you will see that knowledge of N is not necessary, that φ0 and S0 when used together are sufficient to estimate process parameters.
  1. Use the two expressions f and g from the previous sections to eliminate N producing an expression that relates μ, σ, S0, and φ0, an expression of the form S0 = h(μ, σ, φ0)
We will now use this relationship to estimate the parameter α of an exponentially-distributed random process without knowing N. We start with a 1.5 second sample of an exponentially-distributed random process p(x) = αe–αx. The parameter α is randomly chosen from the set {0.1, 0.2, 0.4, 0.8, 1.6, 3.2, 6.4, 12.8, 25.6}. By pressing the “retry” button below () another sample of the process will be generated but with a new, probably different value for the parameter α.
  1. Using the expression h from above and the measured values S0 and φ0, estimate the value of α that was used to generate this sample. Call this estimate αh.

  2. By what percentage does αh differ from the true value α?

  3. Now switch from exponential noise to Poisson noise using the “radio button” below.
We continue with a 1.5 second sample of a Poisson-distributed random process p(x) = λne–λ/n!. The parameter λ is randomly chosen from the set {0.1, 0.2, 0.4, 0.8, 1.6, 3.2, 6.4, 12.8, 25.6}. By pressing the “retry” button below () another sample of the process will be generated but with a new, probably different value for the parameter λ.
  1. Using the expression h and the measured values S0 and φ0, estimate the value of λ that was used to generate this sample. Call this estimate λh.

  2. By what percentage does λh differ from the true value λ?
